# CHRONOS-
​Chronos: A minimalist, high-performance personal ritual tracker built with React and Dexie.js. Features local-first data encryption and Dark aesthetics. Simple and useful for tracking daily habits! 


> A private ritual tracker built for my personal daily growth. Uses React and local storage to keep all my data and photos strictly on my device.


### Private Daily Tracker
Chronos is my own space for tracking daily habits and personal growth. I built this specifically for my private use because I wanted a tracker that matches my aesthetic and keeps my data off the internet.
### Personal Focus
I designed this to be highly personalized for my own lifestyle. It helps me stay consistent with my workouts, studies, and personal rituals without the distraction of a typical app.
### How it Works
As an engineering student, I chose to build this using React and Dexie.js so that everything is stored locally on my phone. This means my photos and diary entries never leave my device and stay completely private.
### Features
The app uses a 30-day grid to show my progress at a glance. It includes a private photo vault for tracking physical changes and a diary for my thoughts. It is simple, quiet, and exactly what I need for my daily routine.
